## Quick Sample — Experienced (Hinglish)

> *"Hi, mera naam Swati Singh hai. Mera total IT experience around [X years] ka hai, jisme se [Y years] main DevOps Engineer ke role mein kaam kar raha hu.*
>
> *Abhi main ek client ke saath kaam kar raha hu jo [domain] mein hai. Client ne ek platform banaya hai jaha users [simple use case] kar sakte hain.*
>
> *Team structure ki baat karu to, approximately [X] frontend engineers, [X] backend engineers, aur [X] DevOps engineers hain.*
>
> *Meri main responsibility hai infrastructure aur CI/CD pipelines ko manage karna using Terraform on Azure aur Azure DevOps.*
>
> *Terraform mein mujhe state file, remote state, state locking, implicit aur explicit dependencies par hands-on experience hai.*
>
> *So overall, meri role hai deployments smooth rakhna aur system reliable rakhna."*

## Quick Sample — Fresher (Hinglish)

## How to Customize

### Ye Karein

- Saare placeholders apni details se replace karo
- Sirf wahi tools mention karo jo aapne **sach mein use** kiye hain
- Length **60 to 90 seconds** rakhe
- Simple language use karo

### Ye Mat Karein

- Ratna mat — robotic lagega
- Project ko overcomplicate mat karo
- Jo tool use nahi kiya, uska naam mat lo
- Ek lambi saans mein sab mat bolo
- Abruptly mat khatam karo
